# Language Translation Tool

A **web-based Language Translation Tool** that translates text between multiple languages in real-time. Built with a **HTML/CSS/JS frontend** and a **Python Flask backend** using `deep-translator` (GoogleTranslator).

---

## Features

- Translate text between **13+ languages**: English, Spanish, French, German, Italian, Portuguese, Russian, Japanese, Korean, Chinese (Simplified), Arabic, Hindi, Urdu.  
- **Text-to-Speech** support for translations.  
- **Copy translation** to clipboard with one click.  
- **Responsive and user-friendly interface**.  
- **Error handling** for empty input, same source/target languages, and API errors.  
- **Health check endpoint** to ensure the API is running.  

---

## Tech Stack

- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** Python, Flask  
- **API Integration:** `deep-translator` (GoogleTranslator)  
- **CORS Support:** `flask-cors` for frontend-backend communication
