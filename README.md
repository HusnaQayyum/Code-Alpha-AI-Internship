# Code Alpha AI Internship

This repository contains all projects and tasks completed during my **AI Internship at Code Alpha**, focusing on practical AI/ML applications. Projects include building **chatbots**, **FAQ systems**, and **NLP pipelines** with Python, NLTK, and spaCy, along with **front-end integration** using HTML, CSS, and JavaScript. The repo demonstrates hands-on AI/ML skills, data preprocessing, text similarity matching, and real-world solution deployment.

---

## Skills Gained
- Python programming for AI/ML  
- Natural Language Processing (NLP) with **NLTK** and **spaCy**  
- Building AI chatbots and FAQ systems  
- Data preprocessing and cleaning techniques  
- Text similarity matching using **cosine similarity** and **intent matching**  
- Front-end integration with AI solutions (**HTML, CSS, JavaScript**)  

---

## Projects

### 1. FAQ Chatbot
**Objective:** Build a chatbot to answer FAQs about a topic or product.  

**Features:**  
- Collects FAQs and answers  
- Preprocesses text: tokenization, cleaning, normalization  
- Matches user questions to FAQs using cosine similarity or intent matching  
- Displays relevant answers in a conversational UI  

**Tech Stack:** Python, NLTK, spaCy, HTML, CSS, JavaScript  

---

### 2. Language Translator Tool
**Objective:** Build a tool to translate text between multiple languages.  

**Features:**  
- Translate text from a source language to a target language  
- Supports multiple languages  
- Simple and interactive front-end interface  
- Integrated with Python backend for processing  

**Tech Stack:** Python, HTML, CSS, JavaScript
